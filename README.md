# Personal Jarvis Installer

> Разверни на своём VPS персонального AI-агента с памятью, скиллами и Telegram-чатом за 2 часа.

Открытый инструктаж по созданию AI-ассистента типа Jarvis на собственном
сервере. Никаких SaaS, никаких чужих серверов, никакой подписки за
«агентскую платформу». Только Claude Code, Python и ваш VPS.

## Что вы получите

После прохождения инструкции:

- ✅ AI-агент на вашем сервере (~4€/мес VPS)
- ✅ Память 4 уровня (моментальный контекст + долгосрочный архив)
- ✅ Cron-ротации памяти (автоматическая компрессия, архивация)
- ✅ Telegram-бот: пишете в личку, отвечает текстом или голосом
- ✅ Allowlist (никто кроме вас не общается с ботом)
- ✅ Распознавание голосовых сообщений (через Groq Whisper)
- ✅ Базовый набор скиллов: brainstorm, present, voice, reminders
- ✅ MCP n8n-mcp (1505 нод n8n как справочник для автоматизаций)
- ✅ Systemd: всё перезапускается само после ребута

**Под капотом:** Claude Code (Anthropic) + Python Telegram Bot +
кастомные cron-скрипты + 4-слойная архитектура памяти.
